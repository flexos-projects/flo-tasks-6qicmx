# Features Overview

## Core capabilities

1. **Client management** — contacts, briefs, communication history, status tracking
2. **Project management** — scoped to a client, with budget, deadline, brief
3. **Task tracking** — tasks within projects, status board, assignees
4. **Time tracking** — log time against tasks or projects, running timer
5. **Invoice generation** — auto-build from time logs, customisable, PDF export
6. **Invoice tracking** — sent/viewed/paid status, automated payment reminders
7. **Dashboard** — live overview of active work, unbilled hours, overdue invoices
8. **Reporting** — revenue by client/month, utilisation rate, top clients

## What we deliberately don't do

- No team chat (use Slack)
- No file storage (use Dropbox/Drive, just link)
- No accounting/tax (use Xero, we export)
- No client portal v1 (roadmap)

## Feature philosophy

Ship the workflow end-to-end before adding depth. A freelancer should be able to onboard a new client, log their first hour, and send their first invoice on day one — before we add power features like retainers, recurring invoices, or team billing.
