# Flows Overview

## Core flows

1. **Client onboarding** — first contact to first project created (5 steps)
2. **Project lifecycle** — brief to delivery, tasks and time throughout (6 steps)
3. **Invoice flow** — generate from time entries to payment confirmed (5 steps)
4. **Freelancer onboarding** — signup to first invoice sent (the demo flow for new users)

## Design principles

- Every flow ends at a clear outcome — a project created, an invoice sent, money received
- No dead ends — every screen has a clear next action
- Time logging is the atomic action — it should be so fast it becomes habit
