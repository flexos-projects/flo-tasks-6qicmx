# Project Origin

**Created as:** FlexOS demo project

**Original prompt:** "Build a client and project management tool for freelancers — from first client brief through to sending and tracking invoices. Needs to handle multiple clients, projects within clients, tasks within projects, time tracking, and invoice generation."

**Why this product:** Freelancers spend 20–30% of their time on admin. Chasing invoices, tracking hours across projects, keeping clients updated. Tools like Notion are too flexible, FreshBooks too accountant-focused. Flo is opinionated for the freelancer workflow.

**Target user:** Independent freelancers — designers, developers, writers, consultants — billing 3–10 clients at any time.
